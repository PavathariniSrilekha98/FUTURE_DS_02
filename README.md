# Customer Retention & Churn Analysis Dashboard

## 📊 Project Overview
This project analyzes customer churn patterns, retention trends, and revenue impact using Microsoft Excel. The objective is to identify churn drivers and provide actionable business insights to improve customer retention.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design

## 📈 Key Metrics
- Total Customers: 50
- Churn Rate: 34%
- Retention Rate: 66%

## 📊 Analysis Performed
- Churn by Subscription Plan
- Monthly Churn Trend
- Churn by Payment Method
- Active Customer Trend
- Revenue Lost Due to Churn
- Revenue by Subscription Plan
- Customer Status Overview (Active vs Churned)
- Customer Distribution by Plan

## 🔍 Key Insights
- Basic subscription plan shows the highest churn rate.
- Revenue loss is concentrated in specific months.
- Certain payment methods are linked with higher churn.
- Overall retention rate remains stable at 66%.

## 📌 Recommendations
- Improve features and value of the Basic subscription plan.
- Introduce retention campaigns during high churn months.
- Encourage digital payment methods.
- Strengthen early customer engagement strategies.
